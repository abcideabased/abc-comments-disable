Disable the WordPress built-in comment system.

== Description ==

ABC Comment Disable removes the built-in comment system from the WordPress back-end as well as the front-end.

Major features in include:

* Disable support for comments and trackbacks in post types.
* Close comments on the front-end.
* Hide existing comments.
* Remove comments page in menu.
* Redirect any user trying to access comments page.
* Remove comments metabox from dashboard.
* Remove comments links from admin bar.
* Remove from admin bar.
* Remove comments from At a Glance.
* Disable comments RSS feed and add error notice.

== Installation ==

Upload the plugin to your blog and activate it.

1, 2, 3: You're done!
